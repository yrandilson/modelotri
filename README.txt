Pasta de assets — Projeto Luminária Balança RA
==================================================

Este projeto utiliza apenas primitivas geométricas do A-Frame,
sem arquivos externos de modelos 3D.

Formas usadas no index.html:
- a-cylinder: base e hastes
- a-box: braco horizontal
- a-cone: abajures
- a-sphere: lampadas (emissive)

Bibliotecas via CDN:
- A-Frame 1.6.0: https://aframe.io/releases/1.6.0/aframe.min.js
- AR.js: https://raw.githack.com/AR-js-org/AR.js/master/aframe/build/aframe-ar.js
